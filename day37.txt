=========================================
Task 1: Theory Answers
=========================================

1. What is Regression?

Regression is a supervised machine learning technique used to predict continuous numerical values. The output is always a number.

Examples:
- House price prediction
- Salary prediction
- Temperature prediction
- Sales forecasting

2. What is Classification?

Classification is a supervised machine learning technique used to predict categories or labels. The output belongs to a specific class.

Examples:
- Spam or Not Spam
- Pass or Fail
- Disease or No Disease
- Cat or Dog

3. Difference Between Regression and Classification

| Regression | Classification |
|------------|---------------|
| Predicts numerical values | Predicts categories |
| Continuous output | Discrete output |
| Example: Salary Prediction | Example: Spam Detection |
| Output is a number | Output is a label |

4. What is Continuous Data?

Continuous data can take any numerical value within a range.

Examples:
- Height
- Weight
- Salary
- Temperature

5. What is Categorical Data?

Categorical data is divided into groups or categories.

Examples:
- Male/Female
- Pass/Fail
- Spam/Not Spam
- Yes/No

=========================================
Task 6: ML Logic Thinking
=========================================

1. Explain why salary prediction is regression.

Salary prediction is a regression problem because the output is a continuous numerical value. Salaries can have many possible numeric values, so regression algorithms are used.

2. Explain why spam detection is classification.

Spam detection is a classification problem because emails are classified into categories such as Spam or Not Spam. The output is a label rather than a number.

3. Compare Output Types

Regression Output:
- Numerical values
- Continuous data
- Examples: Salary, Marks, Temperature

Classification Output:
- Categories or labels
- Discrete data
- Examples: Pass/Fail, Spam/Not Spam, Disease/No Disease

=========================================
Conclusion
=========================================

Regression predicts continuous numerical values.
Classification predicts categories or labels.

Numeric Output → Regression
Category Output → Classification
