TASK 1: THEORY ANSWERS

1. What is Machine Learning?

Machine Learning (ML) is a branch of Artificial Intelligence that enables computers to learn from data and make predictions without being explicitly programmed.

--------------------------------------------------

2. Why is ML Important?

Machine Learning is important because:

• It automates tasks
• It predicts future outcomes
• It analyzes large amounts of data quickly
• It improves decision-making
• It helps build intelligent systems

--------------------------------------------------

3. What is a Dataset?

A dataset is a collection of data used for training and testing machine learning models.

--------------------------------------------------

4. Difference Between AI and ML

Artificial Intelligence (AI):
• Makes machines intelligent
• Uses rules and reasoning
• Broader concept

Machine Learning (ML):
• Subset of AI
• Learns from data automatically
• Makes predictions based on patterns

--------------------------------------------------

5. Types of Machine Learning

1. Supervised Learning
2. Unsupervised Learning
3. Reinforcement Learning

3. Reward/Penalty Explanation

Reward:
Positive feedback given when the model performs a correct action.

Penalty:
Negative feedback given when the model performs an incorrect action.

The goal is to maximize rewards and minimize penalties.

     ↓

     ↓

     ↓

     ↓

--------------------------------------------------

2. Explain Each Step

Collect Data:
Gather data from different sources.

Clean Data:
Remove errors, duplicates, and missing values.

Train Model:
Teach the model using training data.

Test Model:
Evaluate the model's performance.

Prediction:
Use the trained model to predict outcomes for new data.

--------------------------------------------------
